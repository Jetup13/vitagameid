# Link

[Vita HTML link](https://jetup13.github.io/vitagameid/vita.html)

# How-to use

1. Open [link](https://jetup13.github.io/vitagameid/vita.html)
2. Search name of game/gameid
3. Select ES-DE (.psvita) or Daijishou (.dpt)
4. Generate zip

# ES-DE

* Unzip files inside `~ROMs/psvita/` folder

# Daijishou 

* Unzip files inside a folder
* Sync folder with Daijishou's Vita platform
